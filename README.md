
# Kanban Board

Projeto criado com React para reproduzir o comportamento de um quadro kanban.


## Como rodar: 

    git clone https://github.com/luancma/task-board.git
    cd task-board
    yarn
    yarn start


## Ferramentas utilizadas: 

 - React 
 - React Context
 -  React-beautiful-dnd
 - Styled-Components
 - Grommet
 - Mirage.js


## Features
- Criação de tasks
 - Mover tasks entre as colunas
- Remoção de tasks
- Soft Delete
